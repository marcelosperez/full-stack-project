# Full Stack Development - Learning & Projects

This repository serves as a central hub for my journey into Full Stack Development. It contains a collection of practical exercises, course projects, and personal experiments focused on modern web technologies.

## 🚀 Technologies & Tools

Throughout these projects, I have been working with a variety of languages and frameworks:

*   **Frontend:** HTML5 (Semantic), CSS3 (Flexbox, Grid), JavaScript (ES6+), and jQuery.
*   **Backend:** Node.js and Express.js.
*   **Templating & Logic:** EJS (Embedded JavaScript templates) and Middleware integration.
*   **Data & APIs:** RESTful APIs, JSON, and Axios for server-side requests.

## 📂 Repository Structure

The folders are organized by topic, following my learning path:

| Folder Range | Key Topics Covered |
| :--- | :--- |
| **00 - 08** | HTML5, CSS basics, Design principles, and Responsiveness. |
| **09 - 11** | Advanced CSS (Flexbox/Grid) and Bootstrap. |
| **19 - 24** | jQuery, Backend introduction, and Custom Middleware. |
| **25 - 28** | EJS, API Integration (Axios), and Building REST APIs. |

## 🌟 Project Highlights

*   **REST API Development:** Building and documenting full CRUD operations.
*   **Authentication & Security:** Implementing secrets access and server-side logic.
*   **Responsive Web Design:** Creating mobile-first layouts like the 'Pricing Table' and 'TinDog'.

## 🛠️ How to run

1. Clone this repository: 
   `git clone https://github.com/marcelosperez/FullStackProjects.git`
2. Navigate to the desired folder:
   `cd [folder-name]`
3. Install dependencies (if applicable):
   `npm install`
4. Run the project:
   `node index.js` or open the `index.html` file
